# Работа в проекте School с панелью администратора


Добавлено
--------------------
* Ученики = деделаны;
* Учителя = доделаны;
* Расписание = доделаны;


Не добавлено
---------------------
* Ученики = не доделаны;
* Классы = не доделаны;

Что делать далее
---------------------
Основыные задачи:
  1) Доделать учеников
  2) Доделать классы
  3) Добавить недостающие страницы
  4) Оптимизировать панель администратора
  5) На главное окно добавить все удаленные записи(Если запись Activ = false)
